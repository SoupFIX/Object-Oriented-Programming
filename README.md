# HI Everyone
## This repo consist of Working with Oops Concepts 
## Programming Language : JAVA
## Software Requirements
## 1. 🖥️ Java Development Kit (JDK)

#### JDK 8 or above (JDK 17 or 21 LTS recommended)
#### Download: oracle.com/java or adoptium.net
#### Includes the Java Compiler (javac) and Java Runtime Environment (JRE)

## 2. 🌍 Environment Variable Setup

#### Set JAVA_HOME pointing to your JDK installation folder
#### Add JDK's bin folder to your system PATH variable
#### Verify installation by running in terminal:
 ##### java -version

 ### Any IDLE or Code Editor
 ## Notepad++
 ### 🔽 Step 1: Install Notepad++
#### 1. Go to [notepad-plus-plus.org](https://notepad-plus-plus.org/downloads/)
#### 2. Download the latest version (**.exe installer**)
#### 3. Run the installer → Click **Next → Agree → Install → Finish**

## ⚙️ Step 2: Set Up Java Syntax Highlighting in Notepad++
#### 1. Open Notepad++
#### 2. Go to **Language** menu (top bar)
#### 3. Scroll down and select **Java**
### 4. Your `.java` files will now have **color-coded syntax** automatically
